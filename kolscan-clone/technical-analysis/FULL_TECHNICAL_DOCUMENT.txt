EXECUTIVE SUMMARY
================================================================================

Kolscan is a Solana analytics platform that monitors high-value wallets
("KOLs") in real-time, calculates trading performance, and displays ranked
leaderboards.

KEY FINDINGS:
-------------
 Uses Helius webhooks for real-time data + periodic polling
 Pre-computed aggregations with 5-15 minute refresh cycles
 Heuristic-based trade classification (not perfect parsing)
 Multi-layer caching: PostgreSQL → Redis → CDN
 Trades accuracy for speed (~95% accurate at 5% of cost)
 Estimated cost: $200-600/month infrastructure

TECH STACK (INFERRED):
----------------------
- Frontend: Next.js 14+ (React Server Components)
- API: Next.js API Routes / Express.js
- Database: PostgreSQL 14+ with TimescaleDB
- Cache: Redis 7+
- Indexer: Helius Enhanced Transactions API
- RPC: Helius dedicated endpoint
- Queue: BullMQ
- Hosting: Vercel (frontend) + AWS (backend)
